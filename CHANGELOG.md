# Changelog

All notable changes to **Yojana Mitra** will be documented in this file.

The format is based on Keep a Changelog.

## [Unreleased]

### Added
- Offline AI assistant for Indian Government schemes.
- Document upload and processing.
- Hybrid Retrieval-Augmented Generation (RAG).
- Multilingual support (English, Telugu, Hindi).
- Speech-to-text using Faster-Whisper.
- Structured scheme information extraction.
- Streamlit-based user interface.

### Improved
- Retrieval accuracy using hybrid search.
- Telugu localization.
- Chat interface and user experience.
- Answer grounding using retrieved document context.

### Fixed
- Improved multilingual transcription.
- Reduced hallucinated responses for unrelated queries.
