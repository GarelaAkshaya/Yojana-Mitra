# Code of Conduct

## Our Pledge

We are committed to creating a welcoming, respectful, and collaborative environment for everyone contributing to Yojana Mitra.

## Expected Behavior

- Be respectful and inclusive.
- Provide constructive feedback.
- Write clean and well-documented code.
- Review pull requests professionally.
- Help improve the project for all users.

## Unacceptable Behavior

- Harassment or discrimination.
- Offensive language.
- Personal attacks.
- Deliberately submitting malicious code.
- Sharing sensitive information without permission.

## Enforcement

Project maintainers are responsible for ensuring this Code of Conduct is followed and may remove contributions that violate these guidelines.
