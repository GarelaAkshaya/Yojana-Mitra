# Security Policy

## Supported Versions

Yojana Mitra is currently under active development. Security issues reported against the latest version will be investigated.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately to the project maintainers.

Please include:

- Description of the issue
- Steps to reproduce
- Impact
- Screenshots or logs (if applicable)

Please do **not** disclose security vulnerabilities publicly until they have been reviewed.

## Scope

Examples of security issues include:

- Sensitive information exposure
- Prompt injection vulnerabilities
- Unauthorized file access
- Remote code execution
- Dependency vulnerabilities
- Authentication or authorization issues (if introduced in future versions)

We appreciate responsible disclosure and will address confirmed issues as quickly as possible.
